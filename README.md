Tokyo Amesh for Chrome Extension
==========================

Unofficial [Tokyo Amesh](http://tokyo-ame.jwa.or.jp/) Chrome Extension.

## Development

### Requirements

* Grunt
* Bower

### Getting Started

```
npm install
bower install
grunt debug
```

Open Chrome and go to Tools > Extensions > ``Load unpacked extensions...`` and select ``app`` directory.
